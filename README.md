# Python Excel Cleaner

This Python script automates Excel file cleaning.

## Features
- Reads Excel files
- Removes empty rows
- Removes duplicate rows
- Saves a cleaned Excel file

## Requirements
- Python 3
- pandas
- openpyxl

## How to Run
1. Place `input.xlsx` in the same folder
2. Run:
   python excel_cleaner.py
3. Get `cleaned_output.xlsx`

## Use Case
Useful for data cleaning, reporting, and automation tasks.
